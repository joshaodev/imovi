## 🚀 Começando

Para execução:

* Fazer o download ou o clone do projecto.
* Abrir o arquivo index.html


### 📋 Pré-requisitos

Para uma experiência identica ao desenvolvedor, ter instalado no sistema:

* O navegador Google Chrome, na versão 73.0 ou superior
* A font-family "Lato"
